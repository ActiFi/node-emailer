
Sits atop nodemailer.

## Install

In package.json, under dependencies, you can do...

```"emailer": "https://github.com/wookets/node-emailer/0.1.0"```

## Usage

```
var emailer = require('emailer');

emailer.send('SOME:URL');

emailer.send('SOME:URL', callback);


```
